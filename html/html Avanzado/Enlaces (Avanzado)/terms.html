No se realizan ventas reales, envíos, cobros ni transacciones comerciales de ningún tipo.

El usuario se compromete a utilizar este sitio web de forma responsable y únicamente con fines de visualización y prueba.

  • No usar el sitio con fines ilícitos.
  • No intentar vulnerar la seguridad del sitio.
  • No copiar o redistribuir el contenido con fines comerciales.

3. No recopilación de datos personales

Este sitio web NO recopila, almacena ni procesa ningún tipo de dato personal.

  • No se solicitan datos personales como nombre, correo electrónico, dirección o teléfono.
  • No se solicita información de pago.
  • No se utilizan formularios reales de registro o compra.
  • No se emplean cookies de seguimiento, analíticas ni publicitarias.
  • No se comparte información con terceros.

El uso del sitio es completamente anónimo.
